# Credit Card Customer Risk Profiling & Churn Analysis

## 📌 Project Overview
This project focuses on analyzing credit card customer data to identify risk profiles and predict customer churn. By analyzing over 20+ financial attributes, the goal is to provide actionable data-driven insights to optimize financial risk management and customer retention strategies.

## 🛠️ Tech Stack & Methodology
* **Core Tool:** Python (Google Colab)
* **Libraries Used:** Pandas (Data manipulation), NumPy (Numerical computations), Matplotlib & Seaborn (Data Visualization).
* **Data Logic (SQL-Based):** Handled advanced data filtering, conditional subsets, and aggregations using Pandas dataframes—practically implementing relational database (SQL) concepts like `SELECT`, `WHERE`, and `GROUP BY` logic within Python pipelines.

## 📊 Key Analytical Insights
1. **Risk Profiling:** Identified specific credit limit brackets and income categories that exhibit the highest risk of attrition.
2. **Feature Analysis:** Analyzed key metrics such as **Credit Utilization Ratio** relative to customer tenure (**Months on Book**) to uncover dropping engagement patterns before a customer churns.
3. **Business Value:** Developed visual trends that allow stakeholders to easily identify at-risk customer segments and formulate proactive retention policies.

## 📁 How to Run
1. Open the `Data_Science_project.ipynb` notebook directly in Google Colab or Jupyter Notebook.
2. The dataset used is the popular bank churners dataset. Due to drive mappings, ensure you place your local dataset path accordingly.
